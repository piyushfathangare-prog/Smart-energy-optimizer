# ⚡ Smart Home Energy Saver

AI-powered energy management system for tracking, predicting, and optimizing home electricity consumption.

---

## 🚀 Quick Start

### **Run the App:**
```bash
# Method 1: Double-click
RUN_APP.bat

# Method 2: Command line
python -m streamlit run smart_ai_app.py --server.port 8502
```

### **First Time Setup:**
```bash
# Install dependencies
pip install -r requirements.txt
```

## 🎯 Features

- **Energy Tracking** - Track 50+ home appliances
- **ML Predictions** - 30-day consumption forecasts
- **Cost Analysis** - Maharashtra tariff calculations
- **Solar Calculator** - ROI and payback analysis
- **12 Visualizations** - Interactive charts
- **Goal Tracking** - Set and monitor energy goals
- **Environmental Impact** - Carbon footprint tracking
